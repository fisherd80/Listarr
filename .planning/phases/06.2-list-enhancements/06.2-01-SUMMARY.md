# Phase 06.2 Plan 01: Top Rated Presets Summary

**One-liner:** Added top_rated_movies and top_rated_tv presets using TMDB top_rated endpoint with 4-hour caching.

## Objective Achieved

Added Top Rated Movies and Top Rated TV presets to the list creation system. Users can now create lists based on TMDB's critically acclaimed content rankings.

## Tasks Completed

| Task | Name | Commit | Key Changes |
|------|------|--------|-------------|
| 1 | Add Top Rated functions to TMDB service | 85c333f | `get_top_rated_movies()`, `get_top_rated_tv()` |
| 2 | Add cached versions to TMDB cache service | d05982e | `get_top_rated_movies_cached()`, `get_top_rated_tv_cached()` |
| 3 | Add Top Rated handling to routes and import service | 59728fe | Wizard and import support |

## Implementation Details

### TMDB Service Functions
Added two new functions in `tmdb_service.py`:
- `get_top_rated_movies(api_key, page)` - Calls `Movie().top_rated()`
- `get_top_rated_tv(api_key, page)` - Calls `TV().top_rated()`

### Caching Layer
Added cached versions in `tmdb_cache.py`:
- `get_top_rated_movies_cached(api_key, page)` - Uses `_popular_cache` (4-hour TTL)
- `get_top_rated_tv_cached(api_key, page)` - Uses `_popular_cache` (4-hour TTL)

### Routes Integration
Updated `lists_routes.py`:
- `list_wizard()` now detects `top_rated_movies` and `top_rated_tv` presets
- `wizard_preview()` fetches preview using cached top rated functions

### Import Service
Updated `import_service.py`:
- `_fetch_tmdb_items()` handles `top_rated_movies` and `top_rated_tv` list types

Rewrite `tmdb_service.py` to use direct HTTP calls instead of tmdbv3api:

1. **Replace imports:**
   - Remove: `from tmdbv3api import TV, Discover, Movie, TMDb, Trending`
   - Add: `from listarr.services.http_client import http_session, DEFAULT_TIMEOUT, API_BASE_TMDB`

2. **Remove _init_tmdb() function** - no longer needed with direct calls.

3. **TMDB API authentication:** Use query parameter `api_key={api_key}` on all requests.

4. **Endpoint mappings:**
   | Function | Endpoint | Method |
   |----------|----------|--------|
   | validate_tmdb_api_key | /3/movie/popular | GET |
   | get_tvdb_id_from_tmdb | /3/tv/{id}/external_ids | GET |
   | get_imdb_id_from_tmdb | /3/movie/{id}/external_ids or /3/tv/{id}/external_ids | GET |
   | get_trending_movies | /3/trending/movie/{week\|day} | GET |
   | get_trending_tv | /3/trending/tv/{week\|day} | GET |
   | get_popular_movies | /3/movie/popular | GET |
   | get_popular_tv | /3/tv/popular | GET |
   | get_top_rated_movies | /3/movie/top_rated | GET |
   | get_top_rated_tv | /3/tv/top_rated | GET |
   | discover_movies | /3/discover/movie | GET |
   | discover_tv | /3/discover/tv | GET |
   | get_movie_details | /3/movie/{id} | GET |
   | get_tv_details | /3/tv/{id} | GET |

5. **Pattern for each function:**
   ```python
   def get_trending_movies(api_key: str, time_window: str = "week", page: int = 1) -> list:
       if not api_key:
           return []
       url = f"{API_BASE_TMDB}/trending/movie/{time_window}"
       params = {"api_key": api_key, "page": page, "language": "en"}
       try:
           response = http_session.get(url, params=params, timeout=DEFAULT_TIMEOUT)
           response.raise_for_status()
           data = response.json()
           return data.get("results", [])
       except requests.exceptions.RequestException as e:
           logger.error(f"Error fetching trending movies: {e}", exc_info=True)
           return []
   ```

6. **For discover_movies/tv:** Pass filters directly as query params:
   ```python
   params = {"api_key": api_key, "page": page, "language": "en"}
   if region:
       params["region"] = region
   if filters:
       params.update(filters)
   ```

7. **For details endpoints:** Return the full response JSON (not wrapped in class):
   ```python
   def get_movie_details(tmdb_id: int, api_key: str) -> dict:
       url = f"{API_BASE_TMDB}/movie/{tmdb_id}"
       params = {"api_key": api_key, "language": "en"}
       response = http_session.get(url, params=params, timeout=DEFAULT_TIMEOUT)
       response.raise_for_status()
       return response.json()
   ```

Keep all function signatures identical to maintain backward compatibility with tmdb_cache.py.

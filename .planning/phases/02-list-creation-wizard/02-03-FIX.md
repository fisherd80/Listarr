  <action>
The fetchPreview() function makes a POST request without the CSRF token. Fix by:

1. In list_wizard.html: Ensure CSRF token meta tag exists in base.html or add it:
   `<meta name="csrf-token" content="{{ csrf_token() }}">`

2. In wizard.js fetchPreview() function: Add CSRF token header to the fetch call:
   ```javascript
   const csrfToken = document.querySelector('meta[name="csrf-token"]')?.getAttribute('content');

   const response = await fetch("/lists/wizard/preview", {
       method: "POST",
       headers: {
           "Content-Type": "application/json",
           "X-CSRFToken": csrfToken,
       },
       body: JSON.stringify({...}),
   });
   ```

This matches the pattern used in other AJAX calls in the codebase (e.g., lists.js toggle).

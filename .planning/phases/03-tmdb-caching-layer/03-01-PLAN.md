Create new file `listarr/services/tmdb_cache.py` with:

1. **TTL constants** (seconds):
   - TTL_TRENDING = 3600 (1 hour)
   - TTL_POPULAR = 14400 (4 hours)
   - TTL_DISCOVER = 21600 (6 hours)
   - TTL_DETAILS = 86400 (24 hours)

2. **Cache instances** (TTLCache with maxsize):
   - _trending_cache: maxsize=100, ttl=TTL_TRENDING
   - _popular_cache: maxsize=100, ttl=TTL_POPULAR
   - _discover_cache: maxsize=500, ttl=TTL_DISCOVER (more filter combinations)
   - _details_cache: maxsize=1000, ttl=TTL_DETAILS

3. **Single lock** for all caches:
   - _cache_lock = threading.Lock()

4. **Helper function** for filter hashing:
   ```python
   def _hash_filters(filters: dict) -> str:
       """Create stable hash from filter dict for cache key."""
       if not filters:
           return "none"
       import hashlib
       import json
       sorted_json = json.dumps(filters, sort_keys=True)
       return hashlib.md5(sorted_json.encode()).hexdigest()[:8]
   ```

5. **Cached wrapper functions** (8 total, matching tmdb_service.py):
   - get_trending_movies_cached(api_key, time_window='week', page=1)
   - get_trending_tv_cached(api_key, time_window='week', page=1)
   - get_popular_movies_cached(api_key, page=1)
   - get_popular_tv_cached(api_key, page=1)
   - discover_movies_cached(api_key, filters=None, page=1)
   - discover_tv_cached(api_key, filters=None, page=1)
   - get_movie_details_cached(tmdb_id, api_key)
   - get_tv_details_cached(tmdb_id, api_key)

   Each function:
   - Generates cache key from function params (NOT including api_key - it's constant per session)
   - Checks cache with lock
   - On miss: calls underlying tmdb_service function, caches result only if non-empty
   - Uses logging.debug for cache hit/miss

AVOID:
- Including api_key in cache key (wastes cache space)
- Caching empty/None results (API errors would persist)
- Creating separate locks per cache (unnecessary for homelab)

Pattern from dashboard_cache.py:
```python
with _cache_lock:
    if cache_key in _cache:
        return _cache[cache_key]

# Fetch outside lock
result = underlying_function(...)

if result:  # Only cache successful results
    with _cache_lock:
        _cache[cache_key] = result
```

**2. import_list()** - Main orchestration function
```python
def import_list(list_id: int) -> ImportResult:
    """
    Import all items from a TMDB list into Radarr or Sonarr.

    Args:
        list_id: Database ID of the List to import

    Returns:
        ImportResult with added/skipped/failed items
    """
```

Implementation flow:
1. Fetch List from database by ID
2. Fetch ServiceConfig for target_service (radarr/sonarr)
3. Fetch MediaImportSettings for target_service
4. Decrypt API key using crypto_utils
5. Resolve import settings via resolve_import_settings()
6. Fetch TMDB results based on list type/filters (use cached functions from tmdb_cache)
7. Route to _import_movies() or _import_series() based on target_service

**3. _import_movies()** - Import movies to Radarr
```python
def _import_movies(
    tmdb_items: list,
    base_url: str,
    api_key: str,
    settings: dict,
    tmdb_api_key: str
) -> ImportResult:
```
- Pre-flight: Call get_existing_movie_tmdb_ids() once
- For each TMDB item:
  - Extract tmdb_id from item
  - If tmdb_id in existing_ids: add to result.skipped with reason='already_exists'
  - Else: lookup_movie(), if not found add to failed, else add_movie()
  - On exception: add to result.failed with error message
- Log summary at INFO level

**4. _import_series()** - Import TV shows to Sonarr
```python
def _import_series(
    tmdb_items: list,
    base_url: str,
    api_key: str,
    settings: dict,
    tmdb_api_key: str
) -> ImportResult:
```
- Pre-flight: Call get_existing_series_tvdb_ids() once
- For each TMDB item:
  - Extract tmdb_id from item
  - Translate to tvdb_id using get_tvdb_id_from_tmdb()
  - If no tvdb_id: add to failed with reason='no_tvdb_id'
  - If tvdb_id in existing_ids: add to skipped with reason='already_exists'
  - Else: lookup_series(), if not found add to failed, else add_series()
  - On exception: add to failed with error message
- Log summary at INFO level
